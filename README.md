# photo-portfolio

вторая версия минималистичного веб-портфолио для фотографий с умной ленивой загрузкой, кастомным интерактивным лайтбоксом и адаптивной цветовой палитрой.

## оптимизации и логика

- динамическое управление памятью: в сетке галереи отображаются исключительно облегченные превью (около 800px), оригиналы высокого разрешения подгружаются «лениво» только при клике на конкретное фото для открытия в полный экран.
- кастомный лайтбокс: написан с нуля без тяжелых внешних библиотек. поддерживает pinch-to-zoom на смартфонах, масштабирование колесиком мыши (scroll zoom) на пк, перетаскивание (pan) и принудительную очистку памяти от DOM-элемента `<img>` при закрытии (для предотвращения утечек оперативной памяти на мобильных устройствах).
- адаптивная тема через ColorThief: при загрузке страницы скрипт выбирает случайный фоновый снимок из категории s-tier, анализирует его цветовую гамму, рассчитывает доминирующий оттенок HSL и плавно перекрашивает заголовок в этот цвет с повышенной насыщенностью.
- интерактивный курсор: плавное следование точки и круга за мышью на десктопах с автоматическим отключением на тачскринах.
