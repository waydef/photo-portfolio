# Photo Portfolio V2

This is the updated and optimized version of the photo portfolio. V1 was deprecated and removed.

## Optimizations
- Dynamic memory management: Loads low-res previews (~800px) in the grid and fetches high-res originals only when clicking on a photo.
- Custom Lightbox: Pinch-to-zoom on mobile, scroll zoom on PC, panning, and GC memory clearing on close.
